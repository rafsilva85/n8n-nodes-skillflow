# 🔗 n8n-nodes-skillflow

n8n community node for [SkillFlow](https://skillflow.builders) — Search, browse, and install AI skills in your n8n workflows.

## Operations

| Operation | Description |
|-----------|-------------|
| Search Skills | Search for AI skills by keyword |
| Get Skill Details | Get details of a specific skill |
| List Categories | List all skill categories |
| Trending Skills | Get trending skills this week |

## Installation

In your n8n instance:
1. Go to **Settings > Community Nodes**
2. Enter `n8n-nodes-skillflow`
3. Click Install
